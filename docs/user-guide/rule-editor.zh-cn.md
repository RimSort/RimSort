# 规则编辑器

要访问规则编辑器，你可以在主菜单栏中点击 `编辑 > 规则编辑器...（Edit > Rule Editor...）`，或者右击想要编辑的 Mod，在上下文菜单中选择 `其他选项 > 在规则编辑器中编辑（Miscellaneous Options > Edit mod with Rule Editor）`。
